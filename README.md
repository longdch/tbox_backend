# TBOX Backend Homework

## Getting Started
### Prerequisites

* Docker compose [Install](https://docs.docker.com/compose/install)
* Go [Install](https://golang.org/doc/install)

## Running the tests

```
make test
```

### Run code

Build containers
```
make build
```

Run containers
```
make run
```

Stop containers
```
make stop
```
